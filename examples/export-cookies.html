Cookie Exporter

Export cookies from your browser to use with AI Browser — no password sharing needed.

Method 1: Browser DevTools (Recommended)

1. Log in to the website you want the agent to access

2. Open DevTools (F12 or Cmd+Option+I)

3. Go to the Console tab

4. Paste this script and press Enter:

(() => {
  const cookies = document.cookie.split(';').map(c => {
    const [name, ...rest] = c.trim().split('=');
    return {
      name: name,
      value: rest.join('='),
      domain: window.location.hostname,
      path: '/',
      secure: window.location.protocol === 'https:',
    };
  });
  const json = JSON.stringify({ cookies, domain: window.location.hostname, exportedAt: new Date().toISOString() }, null, 2);
  const blob = new Blob([json], { type: 'application/json' });
  const a = document.createElement('a');
  a.href = URL.createObjectURL(blob);
  a.download = `cookies-${window.location.hostname}.json`;
  a.click();
  console.log(`Exported ${cookies.length} cookies for ${window.location.hostname}`);
})();

5. A JSON file will download. Use it with AI Browser:

curl -X POST http://localhost:3000/sessions/SESSION_ID/cookies/import-file \
  -H 'Content-Type: application/json' \
  -d '{"filePath":"/path/to/cookies-example.com.json"}'

Method 2: EditThisCookie Extension

1. Install the EditThisCookie browser extension

2. Log in to the target website

3. Click the EditThisCookie icon and choose Export

4. Save the exported JSON to a file

5. Import into AI Browser — the format is auto-detected

Using Exported Cookies with AI Browser

Via MCP (Claude Code):

Use the import_cookies_from_file tool with the session ID and cookie file path.

Via REST API:

# From a file
curl -X POST http://localhost:3000/sessions/SESSION_ID/cookies/import-file \
  -H 'Content-Type: application/json' \
  -d '{"filePath":"/Users/you/cookies-github.com.json"}'

# From raw JSON
curl -X POST http://localhost:3000/sessions/SESSION_ID/cookies/import \
  -H 'Content-Type: application/json' \
  -d '{"data":"[{\"name\":\"session\",\"value\":\"abc123\",\"domain\":\"github.com\"}]"}'

Via JS SDK:

import { AIBrowser } from '@ai-browser/sdk';

const session = await browser.createSession();
await session.setCookies([
  { name: 'session', value: 'abc123', domain: 'github.com' },
]);
await session.navigate('https://github.com/dashboard');

Via Python SDK:

from ai_browser import AIBrowser, CookieParam

session = await browser.create_session()
await session.set_cookies([
    CookieParam(name="session", value="abc123", domain="github.com"),
])
await session.navigate("https://github.com/dashboard")
Security note: Cookie files contain your session tokens. Treat them like passwords. Never commit them to git. Add *.cookies.json to your .gitignore.
